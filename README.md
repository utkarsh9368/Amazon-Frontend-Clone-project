# 🛒 Amazon Clone

A responsive Amazon Clone built using **HTML, CSS, and JavaScript**. This project recreates the look and feel of Amazon's homepage while focusing on responsive web design, clean UI, and interactive frontend features.

## 🚀 Features

- Responsive Design
- Amazon-inspired Homepage
- Navigation Bar
- Search Bar
- Product Categories
- Hero Banner
- Product Cards
- Shopping Cart Interface
- Footer Section
- Mobile Friendly Layout
- Interactive JavaScript Features
